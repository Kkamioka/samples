# 各フォルダの構成について

ソースコードをダウンロードして頂き、ありがとうございます。
各フォルダの構成は、以下のようになっています。



▼ REST API とは

spring-rest1・・・3.1 ドキュメントの作成方法を確認する

spring-rest2・・・3.3 CRUD（クラッド）操作ができるようにする

spring-rest3・・・3.4 CRUDを自作する



▼ MyBatis

spring-mybatis・・・4.1 データを取得して表示する

spring-mybatis-crud・・・4.2 データのCRUD操作ができるようにする

spring-mybatis-xml・・・4.3 SQLをXMLに記述する

spring-mybatis-pagination・・・4.4 ページネーションを実装する



▼ AWS Elastic Beanstalk とは

spring-beanstalk・・・6.1 Elastic BeanstalkでWebアプリケーションを作成する



▼ Amazon RDS

spring-rds・・・8.1 Amazon RDSでWebアプリケーションを動作させる
